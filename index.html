Particle Offline Utility:

My favorite development platforms are the Particle Photon, Electron and Arduino boards. I created and maintain a script for installing and using the Particle Toolchain on Ubuntu-based Distros and OSX. The script simplifies offline development and makes it easy to compile firmware for Photons and Electrons. The script is called po-util, an abbreviation of Particle Offline Utility. The repository for the script can be found at https://github.com/nrobinson2000/po-util. The script is simple to use, and can be easy installed on Linux and OSX with:

curl -fsSL bit.ly/install-po-util | bash
