Projects:

Particle Offline Utility:

My favorite development platforms are the Particle Photon and Electron. I created and maintain a script for installing and using the Particle Toolchain on Ubuntu-based Distros and OSX. The script simplifies offline development and makes it easy to compile firmware for Photons and Electrons. The script is called po-util, an abbreviation of Particle Offline Utility. The repository for the script can be found on GitHub. The script is simple to use, and can be easily installed on Linux and OSX with:

$ git clone https://github.com/nrobinson2000/po-util
$ cd po-util
$ ./po-util.sh install

Read more about po-util at po-util.com

Particle has recently recognized po-util as a great contribution. Read it here.

Nano Control:

Nano Control makes it possible to interact with an Arduino Nano as if it was part of a Particle Device. It uses simple Serial commands to call basic functions on an Arduino Nano. The purpose of Nano Control is to expand the number of usable pins on a Particle device by creating an easy-to-manipulate Arduino slave.

Nano Control consists of two libraries. One called NanoMaster that runs on a Particle Device, and one called NanoSlave that runs on an Arduino Nano.

Read more about Nano Control on GitHub.
