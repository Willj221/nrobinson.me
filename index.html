Particle Offline Utility:

My favorite development platforms are the Particle Photon and Electron. I created and maintain a script for installing and using the Particle Toolchain on Ubuntu-based Distros and OSX. The script simplifies offline development and makes it easy to compile firmware for Photons and Electrons. The script is called po-util, an abbreviation of Particle Offline Utility. The repository for the script can be found on GitHub. The script is simple to use, and can be easy installed on Linux and OSX with:

curl po-util.com/download | bash

Read more about po-util at po-util.com

Particle has recently recognized po-util as a great contribution. Read it here.

Bitcoin Donation Page Generator:
